# GeoMEOL
PredGAMM is a geometry-aware multimodal learning framework for the prediction of secreted effectors in Gram-negative bacteria. The key idea behind PredGAMM is to reformulate effector prediction from the perspective of joint sequence-structure representation.Specifically, PredGAMM first applies an attention mechanism to the embedding vectors generated by the pretrained protein language model ESM-2, so as to further extract contextual semantic information from protein sequences. It then employs a geometric vector perceptron (GVP) to encode local geometric patterns and residue-level spatial relationships from protein structures, enabling the joint modeling of semantic and conformational features. Building on this, PredGAMM further integrates an auxiliary contrastive learning objective with a tokenization-based multi-head attention mechanism to enhance cross-modal alignment between sequence and structure and to model fine-grained multimodal interactions, thereby improving the identification of key effector-related patterns.

## Data Preparation

### 1. FASTA Files

Training and test FASTA files with effector type labels in the header. Supported label patterns:

| Label | Patterns Recognised |
|---|---|
| Non-Effector (0) | `Non-Effector`, `non_effector`, `negative`, `NEG` |
| T1SE (1) | `T1SE`, `T1SS`, `Type1`, `Type 1` |
| T2SE (2) | `T2SE`, `T2SS`, `Type2`, `Type 2` |
| T3SE (3) | `T3SE`, `T3SS`, `Type3`, `Type 3` |
| T4SE (4) | `T4SE`, `T4SS`, `Type4`, `Type 4` |
| T6SE (5) | `T6SE`, `T6SS`, `Type6`, `Type 6` |

### 2. PDB Structure Files

Predicted or experimental structures in PDB format, one file per protein. File names should match the FASTA sequence IDs (e.g., `proteinA.pdb` for sequence ID `proteinA`).

### 3. ESM-2 Embeddings

Pre-computed residue-level embeddings from ESM-2 (36-layer, 3B parameter model), saved as pickle dictionaries mapping protein ID в†’ numpy array of shape `(L, 2560)`.
